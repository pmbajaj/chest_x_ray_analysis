About This Project

This X-Ray Image Classifier is a machine learning model aimed at classifying X-ray images into two categories: Normal and Pneumonia. The model uses a Convolutional Neural Network (CNN) to extract features from images and provide a prediction based on the patterns recognized in the training data.

Technologies Used:
  • Python: The main language used for developing the model and application logic.
  • PyTorch: Utilized for building, training, and testing the CNN model.
  • Flask: Serves as the web framework for hosting the application and handling user requests.
  • SQLAlchemy and SQLite: Used for database management, storing user contact messages securely within the app.
  • Bootstrap: Ensures a responsive and user-friendly interface.

Project Highlights

Some unique aspects of this project:

  • Cross-Validation: To ensure robustness, the model was trained with cross-validation, leveraging multiple folds to find the best performance metrics.
  • Imbalanced Dataset Handling: Techniques like weighted sampling were applied to address class imbalance in the dataset, ensuring the model performs well across both categories.
  • Deployment and Accessibility: Developed with user accessibility in mind, allowing users to upload their X-ray images and receive predictions with a simple interface.
